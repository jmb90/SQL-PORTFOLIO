# Sakila SQL Portfolio

This repository contains a collection of SQL queries written to analyze the
**Sakila sample database** using **PostgreSQL**.

The goal of this project is to demonstrate solid SQL skills, progressing from
basic queries to more advanced concepts such as joins, aggregations, subqueries,
and analytical thinking.

---

## Dataset

- **Database**: Sakila (DVD Rental sample database)
- **DBMS**: PostgreSQL
- **Tooling**: pgAdmin

Sakila represents a video rental store and includes data about films, actors,
customers, rentals, payments, and inventory.
